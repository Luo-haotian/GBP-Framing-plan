# GBP Framing

This case folder stores V2 schema and review artifacts.

Current contents:

- `example-schema-demo/`: synthetic example instance for schema validation and review-DXF generation.
- `q2-process-test-0.11/`: drawing-derived Figure Q2 workflow test using compact standard-floor groups for Revit review.

Important:

- The example schema demo is not a drawing-derived reconstruction of `Figure Q`.
- It exists to validate the V1 neutral structural model contract and the review-DXF generation flow.
- `q2-process-test-0.11` is review-grade only. It records standard-floor compaction, visible member dimensions, opening edge beams, wall-supported slab logic, and Revit import performance safeguards.
