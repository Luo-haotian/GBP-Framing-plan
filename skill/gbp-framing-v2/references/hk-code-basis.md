# HK Code Basis

## Purpose

Use this reference when the GBP framing task is for a Hong Kong project and code-derived assumptions must be stated clearly.

## Expected Local Code Pack

Prefer a local `HK Code` folder when available, including:
- `CoP_SUC2013e.pdf`
- `DIL2011e.pdf`
- `FoundationCode2017.pdf`
- `SUOS2011.pdf`
- `WindEffects2019e.pdf`
- `fs_code2011.pdf`

## What To Pull

- imposed loads and reduction logic
- wind basis and torsion-sensitive triggers
- RC or steel design framework
- fire constraints affecting structural sizing
- foundation implications that affect framing assumptions

## Rule

Do not invent numeric code values.

When a code value is used, record:
- source file
- page
- short description of the requirement

If the formal analysis software will govern the final check, say that clearly instead of implying that the skill alone guarantees compliance.
